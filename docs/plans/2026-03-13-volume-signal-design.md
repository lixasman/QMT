# 分级量能信号设计（入场 Phase2）

## 1. 目标
本设计通过调整入场 Phase2 的量能信号计算方式，提高 588200 在 2025 年 8–10 月行情的参与度，同时尽量控制回撤与交易次数。

## 2. 背景与问题
当前回测显示 2025-09-05 之后入场评分长期低于 0.45，关键原因是 S_volume 多数时间为 0，导致整体评分不足以触发信号。

## 3. 方案概述
仅调整 S_volume 的计算逻辑，从二值 0/1 改为三档 0/0.5/1。评分权重与阈值保持不变，退出逻辑不变。

## 4. 规则细节
1. 价格突破条件保持不变：close_t > resistance。
2. 实体强条件保持不变：body_ratio > 0.5。
3. 放量显著：vol_ratio >= 1.5，S_volume = 1.0。
4. 放量中等：1.2 <= vol_ratio < 1.5，S_volume = 0.5。
5. 其余情况 S_volume = 0.0。
6. 只有同时满足“价格突破 + 实体强”时才考虑 1.0 或 0.5，否则直接 0.0。

## 5. 影响范围
1. 仅修改 entry/signals/volume_break.py 的返回值逻辑。
2. 不修改评分权重、阈值或其他信号计算。
3. 不修改退出与救生衣逻辑。

## 6. 数据流与交互
1. Phase2 仍在盘后以日线 bars + watchlist 计算信号。
2. 仅 S_volume 的数值改变，其他信号与评分逻辑不变。
3. 评分函数 compute_entry_score 不变，阈值仍为 0.45。

## 7. 风险与控制
1. 风险：温和放量的入场增多，可能带来回撤上升。
2. 控制：保留“价格突破 + 实体强”的前置条件，避免纯量能噪声。

## 8. 验证与回测
1. 回测范围：588200，2025-01-01 至 2025-12-31。
2. 对比基线回测目录：output/backtest_588200_vol15。
3. 新回测输出目录与基线并列，便于对照年化与回撤变化。

## 9. 回退策略
若回测结果不理想，直接把 S_volume 逻辑恢复为二值 0/1 即可回退。
